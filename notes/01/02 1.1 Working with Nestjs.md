# Phase 1 - Working with Nestjs

## Install NodeJs
To build with NestJS v11, your environment must meet specific requirements. NestJS v11 requires **Node.js v18.10 or higher**. However, for professional development, it is highly recommended to use the **LTS (Long Term Support)** version (currently **v20** or **v22**).

### **Step 1: Checking current versions**
Open your terminal and run these commands to see what is currently installed:
*   **Node version:** `node -v`
*   **NPM version:** `npm -v`

### **Step 2: Installing/Upgrading to the latest LTS**
If you do not have Node installed, or if your version is below 18.10, you have two primary options:

1.  **The Direct Installer (Standard):** 
    Go to [nodejs.org](https://nodejs.org/) and select the **LTS** version. The installer will automatically update your existing Node and NPM versions.
2.  **The Version Manager (Recommended for Experts):**
    Use a tool like **nvm-windows** (for Windows) or **nvm** (for macOS/Linux). This allows you to install multiple versions of Node and switch between them using a simple command:
    *   `nvm install lts` (Installs the latest LTS)
    *   `nvm use lts` (Switches your active terminal to that version)

### **Step 3: Upgrading NPM**
NPM is bundled with Node, but it often has its own update cycle. To ensure you have the latest stable features and security patches, run:
```bash
`npm install -g npm@latest`
```
## Install Nest CLI globally
The **Nest CLI** is a command-line interface that automates the creation and maintenance of NestJS applications. It ensures that your project follows the recommended folder structure and architectural patterns from day one.
Run the following command in your terminal:
```bash 
$ sudo npm install -g @nestjs/cli
```
*   **The `-g` flag:** This installs the package globally on your system, allowing you to use the `nest` command from any directory.
*   **Verification:** After installation, run `nest --version` to ensure it is installed correctly. For NestJS v11, the CLI version should be **11.x.x**.
